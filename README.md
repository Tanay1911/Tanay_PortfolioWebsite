# Portfolio Website — Tanay Panchal

This is a self-contained static portfolio website (HTML, CSS, JS). Drop these files in a folder and open `index.html` in a browser, or deploy to GitHub Pages / Netlify / Vercel.

Files:
- index.html — main page
- styles.css — styling
- script.js — small client-side interactions
- profile.jpg — your photo (replace)
- resume.pdf — your résumé (replace)
- README.md — this file

How to use
1. Replace `profile.jpg` with your photo.
2. Replace `resume.pdf` with your actual résumé.
3. Update links in `index.html`:
   - GitHub, LinkedIn, Email, and project GitHub URLs.
4. Replace project titles/descriptions with your real projects (up to 4).
5. Deploy:
   - GitHub Pages: push to a repo and enable Pages from the main branch.
   - Netlify/Vercel: drag-and-drop or connect repo for auto-deploy.

Notes
- Contact form uses a mailto fallback (no server). If you want a serverless form (Netlify Forms, Formspree) or a Node/Express example, I can add that.
- Accessibility: images include alt text. Consider adding ARIA attributes and descriptive link text when you finalize content.
